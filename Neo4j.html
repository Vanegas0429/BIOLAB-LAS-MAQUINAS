Características

  • Base de datos NoSQL orientada a grafos.
  • Excelente para representar relaciones complejas.
  • Lenguaje de consulta propio llamado Cypher.
  • Usada en redes sociales, recomendaciones y análisis de grafos.

Ejemplos CRUD

// Insertar registro
CREATE (u:Usuario {nombre: 'Juan Pérez', correo: 'juan@example.com'});

// Crear tabla
CREATE (u:Usuario {nombre: "Laura", edad: 28});

// Consultar
MATCH (u:Usuario) RETURN u;

// Actualizar
MATCH (u:Usuario {nombre: "Laura"})
SET u.edad = 29;

// Eliminar
MATCH (u:Usuario {nombre: "Laura"}) DELETE u;
