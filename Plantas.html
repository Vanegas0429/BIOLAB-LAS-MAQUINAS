Diccionario de datos base de datos Biolab

Nombre: Plantas
Creación: 06/08/2025
Descripción: Plantas Del Laboratorio De Biotecnologia Vegetal
Campo Tipo dato Tamaño Descripcion
ID_Planta INT 1 Codigo unico de las plantas
Especie VARCHAR 20 Registrar la especie de planta
Lote INT 1 Registrar el codigo unico del lote donde se encuentra la planta
Metodo_Cultivo VARCHAR 50 Registrar como se cultiva la planta
Metodo_Propagacion VARCHAR 50 Registrar metodo de propagacion de la planta
Numero_Frasco INT 1 Registrar el codigo unico de frasco donde esta la planta
Frascos_Contaminados ENUM 15 Bacterias/Hongos
Frascos_Desarrollados ENUM 40 Con Desarrollo/Sin Desarrollo
Numero_Endurecimiento INT 20 Registrar valor de resistencia de la planta
Planta_Contaminada ENUM ("Bacterias", "Hongos") Tipo de Contaminacion que tiene la planta.
Planta_Desarrollada ENUM ("Con desarrollo", "Sin desarrollo") Estado de desarrollo de la planta.
