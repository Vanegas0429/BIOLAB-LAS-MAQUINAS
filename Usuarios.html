USUARIOS

Diccionario de datos base de datos Biolab

Nombre: Usuarios
Creación: 02/08/2025
Descripción: Usuario del Laboratorio de Biotecnología Vegetal
Campo Tipo dato Tamaño Descripción
ID_Usuario INT 10 Numero de identificacion unico para cada Usuario
Nombre VARCHAR 50 Nombre del Usuario
Tipo_Documento ENUM ("C.C", "T.I","C.E" "C.C") Nombre del Usuario
Documento INT 15 Numero de documento.
cargo VARCHAR 30 Nombre del cargo
Observaciones VARCHAR 100 Registrar informacion dada por el Usuario
Estado ENUM ("Activo", "Inactivo") Estado de actividad del usuario
Fecha DATE 1 Fecha en la que registra la Usuario
